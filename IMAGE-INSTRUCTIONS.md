# Image Instructions

## Where to add Daniel's photo:
1. Save Daniel's professional headshot as: `daniel-photo.jpg` or `daniel-photo.png`
2. Place it in: `public/images/daniel-photo.jpg`

## Where to add background images:
1. Save background images in: `public/images/`
2. Recommended names:
   - `hero-bg.jpg` (for hero section)
   - `about-bg.jpg` (for about section)
   - `contact-bg.jpg` (for contact section)

## How to use the images:
- The code will automatically look for these files
- Supported formats: .jpg, .jpeg, .png, .webp
- Recommended size for Daniel's photo: 400x480px
- Recommended size for backgrounds: 1920x1080px or larger
